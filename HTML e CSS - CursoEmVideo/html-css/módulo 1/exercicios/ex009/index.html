Lista de Definições

HTML
"Linguagem de Marcação de Hipertexto" é uma linguagem de marcação utilizada na construção de páginas na Web.
CSS
Cascading Style Sheet é uma linguagem de marcação usada para estilizar elementos escritos em uma linguagem de marcação como HTML.
JavaScript
É uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma. Juntamente
com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web.
